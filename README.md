# 工数カレンダー

WBSタスクごとの工数（人日）、開始日、稼働曜日、日本の祝日、会社休日から、連続するタスクの完了予定日を計算する静的Webアプリです。

日本の祝日は祝日法ベースで計算し、2026年・2027年の内閣府公表値と照合しています。将来の法改正や春分日・秋分日の公表差分が出た場合は `app.js` の祝日計算を更新してください。

## 使い方

`index.html` をブラウザで開くと利用できます。

WBSの「貼付」から、Excelコピーのタブ区切り、カンマ区切り、または `タスク名 工数` 形式の複数行データを一括読み込みできます。

ローカルサーバーで確認する場合:

```bash
npm run serve
```

テスト:

```bash
npm test
```
